USER INSTRUCTIONS:
New a MovieRegistrationSystem object from the class MovieRegistrationSystem, then use the method: start() to operate the system.
Step 1) System will load data into ArrayList from the file Movies.txt
Step 2) Press 1 to search user from the list
Step 3) Press 2 or 3 to add/delete user into the list.
Step 4) Press 4 to edit the users from the user list you choose.
Step 5) Press 5 to exit the system
Step 6) System will save in ArrayList into the file myMovie.txt
